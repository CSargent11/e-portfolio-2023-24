Unit 1: Review codio module classes and objects – As python is an object-oriented programming language, almost everything in Python is an object, with its properties and methods. A Class is like an object constructor, or a "blueprint" for creating objects. Within these there are different functions to help the programme run smoothly. A class describes the contents of the objects that belong to it: it describes an aggregate of data fields called instance variables, and defines the operations called methods. An object is an element or instance of a class; objects have the behaviours of their class. The object is the actual component of programs, while the class specifies how instances are created and how they behave.

class Surfboard: def __init__(self, brand, length, width): self.brand = brand # Unprotected variable self._length = length # Protected variable self._width = width # Protected variable def get_length(self): return self._length def set_length(self, length): self._length = length def get_width(self): return self._width def set_width(self, width): self._width = width my_surfboard = Surfboard("ABC Surf Co.", 6.2, 22.5) print("Brand:", my_surfboard.brand) # Accessing unprotected variable print("Length:", my_surfboard.get_length()) # Accessing protected variable using a getter method print("Width:", my_surfboard.get_width()) # Accessing protected variable using a getter method my_surfboard.brand = "XYZ Surfboards" # Modifying unprotected variable my_surfboard.set_length(6.4) # Modifying protected variable using a setter method my_surfboard.set_width(23.0) # Modifying protected variable using a setter method print("Updated Brand:", my_surfboard.brand) print("Updated Length:", my_surfboard.get_length()) print("Updated Width:", my_surfboard.get_width())
Unit 2: Review codio modules class function and class methods – In this lab I learnt a function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function and the function will return data as a result. However, class methods are defined inside a class and are similar to defining a regular function. Comparable to inside an instance method, we use the self keyword to access or modify the instance variables. Same inside the class method, we use the cls keyword as a first parameter to access class variables. Therefore, the class method gives us control of changing the class state.
